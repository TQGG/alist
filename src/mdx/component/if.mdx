# If

## 引入方式

```jsx
import Form, { If } from 'noform';
```

## 使用方式

If是解决联动的利器，使用也非常方便，通过`when`函数来决定是否显示内容 (返回`true`即显示，其他均视为隐藏) 。

如果内容涉及**FormItem**，**NoForm**会自动处理**values的整合**，**校验规则的触发**等。

## 实例演示

> 以下例子当trigger不触发时，username不会被校验，不会被收录在values中

```jsx

<Form>
    <FormItem label="trigger" name="trigger">
        <Switch />
    </FormItem>
    <If when={(values)) => values.trigger === true}>
        {/* 返回的元素必须是单元素，可以是FormItem, 可以是div或其他HTML元素 */}
        <div>
            <FormItem name="username" label="hidName"><Input /></FormItem>
        </div>
    </If> 
</Form>

```
