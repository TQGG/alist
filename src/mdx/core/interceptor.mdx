# 拦截器 interceptor

拦截器能够在设置值之前做自定义的修改, interceptor接受方法，返回值可以是 值 或 promise

```jsx
new FormCore({
    interceptor: {
        money: (rawMoney) => { // 接受值
            return parseInt(rawMoney);
        },
        lazyMoney: async () => { // 接受promise
            const money = await remoteCalculateMoney();
            return money;
        }
    }
})
```
